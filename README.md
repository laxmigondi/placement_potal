# Student Placement Preparation Portal

A web-based platform designed to help students prepare for campus placements through study materials, mock tests, aptitude practice, and performance tracking.

## 🚀 Features

### Student Module
- Student Registration & Login
- Dashboard with personalized overview
- Access Study Materials
- Attempt Timed Mock Tests
- View Test Results & Performance History

### Admin Module
- Secure Admin Login
- Add / Manage MCQ Questions
- Create and Manage Mock Tests
- Monitor Student Performance
- Manage Study Materials

## 🛠 Tech Stack

- **Frontend:** HTML, CSS, Bootstrap, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Server:** XAMPP / Apache
